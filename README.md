# homebrew-tap

Homebrew tap for tools by [yuanchuan](https://github.com/yuanchuan).

## Usage

```bash
brew tap yuanchuan/tap
brew install <formula>
```

## Formulae

| Formula | Description |
|---|---|
| [aivo](https://github.com/yuanchuan/aivo) | Run Claude Code, Gemini, and Codex with any API provider |
